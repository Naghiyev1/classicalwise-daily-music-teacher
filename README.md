# ClassicalWise v1.0

Daily classical music teacher.

## What it does

- One famous classical piece per day
- Audio player with direct audio URL
- Fallback link to recording/source page if audio fails
- Why it is special
- What to listen for
- Why you should know this
- Teacher takeaway
- Library search and period filter
- Saved pieces
- Composer timelines
- Period explanations
- How to listen guide
- Mobile-first PWA-style app

## Important note on audio

This first version uses remote public-domain / freely licensed source links where possible. Direct audio URLs can vary by browser/host. The app is designed with fallback links so broken audio does not break the experience.
